Glosario M–Z

Ir a letras A–N
MNOP QRST UVWX YZ

M - Microservicio

Arquitectura donde la aplicación se divide en pequeños servicios independientes.

N - Namespace

Contenedor que permite organizar y gestionar nombres únicos en programación.

O - Open Source

Software cuyo código fuente está disponible para su estudio y modificación.

P - Python

Lenguaje de programación de alto nivel usado para desarrollo web, ciencia de datos, automatización, etc.

Q - Query

Consulta realizada a una base de datos.

R - Repositorio

Almacenamiento central de código fuente y archivos relacionados.

S - Scrum

Marco ágil para la gestión de proyectos de software.

T - Test Unitario

Prueba que verifica el correcto funcionamiento de una unidad de código.

U - UI

Interfaz de usuario; parte visual con la que interactúa el usuario en una aplicación.

V - Variable

Elemento que almacena datos temporales durante la ejecución del programa.

W - Web Service

Servicio que permite la comunicación entre aplicaciones a través de la web.

X - XML

Lenguaje de marcado usado para estructurar datos.

Y - YAML

Formato de serialización de datos legible para humanos, usado en configuración.

Z - Zero-day

Vulnerabilidad de software desconocida por el proveedor y explotada por atacantes.
