Glosario A-N

Ir a letras M-Z
ABCD EFGH IJKL MN

A - Algoritmo

Secuencia lógica de pasos para resolver un problema.

B - Bug

Error o fallo en el software que provoca un comportamiento incorrecto.

C - Compilador

Programa que traduce código fuente a lenguaje máquina.

D - Depuración

Proceso de encontrar y corregir errores en el software.

E - Entorno de desarrollo

Conjunto de herramientas utilizadas para programar, como editores, compiladores y depuradores.

F - Framework

Estructura reutilizable que facilita el desarrollo de aplicaciones de software.

G - Git

Sistema de control de versiones para seguimiento de cambios en archivos de código.

H - HTML

Lenguaje de marcado para la creación de páginas web.

I - Integración continua

Práctica de desarrollo donde se integran cambios frecuentemente con pruebas automáticas.

J - Java

Lenguaje de programación ampliamente utilizado en desarrollo de software empresarial y móvil.

K - Kubernetes

Plataforma de código abierto para la automatización del despliegue, escalado y operación de contenedores.

L - Librería

Conjunto de funciones o clases reutilizables en un lenguaje de programación.

M - Middleware

Software que actúa como puente entre aplicaciones y servicios.

N - Node.js

Entorno de ejecución para JavaScript del lado del servidor.
